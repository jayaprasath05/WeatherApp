# 🌦 WeatherApp

A simple Android Weather Application developed using **Java** and **Android Studio**. The app fetches real-time weather information from the **OpenWeatherMap API** and displays details such as temperature, humidity, wind speed, weather description, and weather icon for any city.

---

## 📌 Features

- Search weather by city name
- Displays real-time weather information
- Shows:
  - Temperature
  - Humidity
  - Wind Speed
  - Weather Description
  - Weather Icon
- Uses OpenWeatherMap API
- Background network requests using ExecutorService
- JSON parsing
- Simple and responsive user interface

---

## 🛠 Technologies Used

- Java
- Android Studio
- Android SDK

## 🚀 How to Run

1. Clone the repository.

```
git clone https://github.com/your-username/WeatherApp.git
```

2. Open the project in Android Studio.

3. Sync Gradle.

4. Replace the API key with your own OpenWeatherMap API key if required.

5. Run the application on an emulator or Android device.
